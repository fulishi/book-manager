# Book Manager

基于 Java Struts2 和 MySQL 的图书管理系统

演示：[点击查看](http://app.codebelief.com/book-manager/)

## 简介

这是一个简化的图书管理系统，包含如下功能：

- 列出所有图书
- 根据作者/书名查询图书
- 查看图书的详细信息
- 添加图书
- 删除图书
- 更新图书
- 添加作者
